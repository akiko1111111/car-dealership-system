# Система учета автомобилей в автосалоне

## Описание проекта
Разрабатываемая информационная система предназначена для автоматизации процессов учета и продажи автомобилей в автосалоне.

## Цель системы
Повышение эффективности работы за счет четкого разграничения обязанностей между покупателями и менеджерами, отслеживания актуального статуса автомобилей и упрощения процедуры бронирования и оформления продаж.

## Роли пользователей

### Покупатель
Пользователь, целью которого является выбор и бронирование автомобиля. Покупатель имеет возможность просматривать каталог, но не может вносить изменения в данные системы.

**Функциональность:**
- Просмотр каталога автомобилей
- Бронирование автомобилей
- Просмотр статуса бронирований

### Менеджер
Сотрудник автосалона, который управляет автомобилями, бронированиями и оформляет продажи. Менеджер обладает расширенными правами для изменения статусов автомобилей и работы с базой данных.

**Функциональность:**
- Управление автомобилями (добавление, редактирование)
- Подтверждение и отмена бронирований
- Оформление продаж
- Просмотр списка всех бронирований

## Функциональность системы

### Для Покупателя
1. **Авторизация/Регистрация** в системе
2. **Просмотр каталога автомобилей**
3. **Выбор автомобиля** с автоматической проверкой статуса
4. **Бронирование** с вводом контактных данных
5. **Подтверждение брони** с изменением статуса автомобиля на "Забронирован"

### Для Менеджера
1. **Авторизация** в системе
2. **Просмотр новых бронирований**
3. **Подтверждение/отмена бронирований**
4. **Добавление новых автомобилей** в каталог
5. **Оформление продаж** с изменением статуса автомобилей

## Статусы автомобилей
- **Доступен** - автомобиль на складе, готов к бронированию
- **Забронирован** - автомобиль забронирован покупателем
- **Продан** - автомобиль продан, снят с продажи

## Преимущества системы
- **Прозрачность:** Все участники процесса видят актуальный статус каждого автомобиля
- **Эффективность:** Сокращается время на обработку бронирований и продаж
- **Удобство:** Покупатели могут самостоятельно бронировать автомобили онлайн
